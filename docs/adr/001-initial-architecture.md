# ADR 001: Initial Architecture

## Status
Accepted

## Context
We need a robust multi-agent architecture to manage software development.
Aizen-Gate was chosen to enforce the Scrum framework.

## Decision
Adopt Aizen-Gate state schema (project.md, memory.md, state.md).

## Consequences
Pros: AI agents have standardized context.
Cons: Requires maintaining markdown files alongside code.
