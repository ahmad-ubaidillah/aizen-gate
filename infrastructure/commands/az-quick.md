# Command: az-quick

The high-speed, atomic implementation phase.

**[SA] Time for a quick-fix!** I'm calling in @quick-flow.

## 1. Quick Discovery (Via [SA] Bob)

- **[SA] Bob** identifies the small task at hand:
  - Is it a bugfix, minor feature, or config change?
  - Does it have clearly defined Acceptance Criteria (AC)?
  - Is it atomic (one commit)?

## 2. Fast-Track Execution (Via [QUICK] Dash)

- **[QUICK] Dash** implements the change:
  - Skip formal PRD and multi-agent debate.
  - Apply the fix/feat directly to the code.
  - Ensure standard unit tests still pass.

## 3. Immediate Verification (Via [QA] Argus)

- **[QA] Argus** performs a rapid check:
  - Run the failed test (if bugfix) or new test (if feature).
  - Verify overall project health.
  - Finalize the change and commit.

---

**[SA] Done in record time!** The atomic task is complete and verified. Ready for the next quick-fix?
