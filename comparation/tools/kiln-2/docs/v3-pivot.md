# Kiln v3 Pivot

Kiln v3 is a shift from orchestrator-heavy behavior to Claude Code native teams.

## Why the Pivot

- Keep coordinator context lean and durable.
- Move information flow to files and teammate-to-teammate collaboration.
- Preserve persistent expertise through long-lived minds (Visionary, Architect, Sentinel).
- Plan at the right time (JIT for implementation phases).

## v2 vs v3

- v2: file pipeline with stronger coordinator ownership.
- v3: plugin-first model with teammate spawning, direct loops, and strict ownership boundaries.

## v3 Location

- `./kiln-v3`

Use it with:

```bash
claude --plugin-dir /path/to/kiln/kiln-v3
```
