# Skill: Professional Document Generation (README)

## Overview

The [DOCS] agent uses this skill to create "premium" documentation that onboards users in minutes.

## README Structure

1. **Brand Identity**: Logo (or ASCII), One-liner, Badges.
2. **Feature Highlights**: Bullet points with emojis.
3. **Architecture**: ASCII diagram or Mermaid.
4. **Getting Started**: 2-3 steps maximum.
5. **API & CLI**: Table of commands and parameters.
6. **Contribution**: Pointer to `CONTRIBUTING.md`.

## Quality Check

- **No Placeholders**: Every command must be working.
- **Accurate Paths**: Verify relative file paths.
- **Clarity & Brevity**: Minimize jargon; maximize insight.
