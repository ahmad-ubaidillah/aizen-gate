# Command: za-help

Welcome to the **Aizen-Gate [SA] Help**! Here are the core commands to drive the framework:

## Core Lifecycle Commands

- `za-start`: Initial onboarding. Checks for PRDs and calibrates the tech stack.
- `za-idea`: Groom your raw idea into a refined concept and PRD with @analyst.
- `za-plan`: Break down the PRD into atomic tasks on the **Scrum Board** with @architect.
- `za-build`: Execute development tasks with @developer and @designer. TDD-driven.
- `za-docs`: Generate comprehensive API, user, and readme documentation with @tech-writer.

## Utility & Review Commands

- `za-verify`: Run security audits (@security) and QA reviews (@qa).
- `za-debate`: Explicitly trigger a debate between agents on a specific trade-off.
- `za-party`: All agents review the current state together.
- `za-quick`: Solo developer flow for smaller, non-scrum tasks.
- `za-status`: Check the current status of the Scrum Board and Memory.

## Extensibility Commands

- `za-add-skill [url]`: Scrape a URL and generate a new Aizen-Gate SKILL.md.
