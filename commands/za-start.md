# Command: za-start

The onboarding command for new projects.

**[SA] Welcome to Aizen-Gate!** I will guide you from idea to deployment.

## 1. Context Collection

First, I need to understand what we're building.

- **[SA]** Do you already have a **PRD (Product Requirements Document)**?
- **[SA]** If **YES**, please point me to the file or paste the contents.
- **[SA]** If **NO**, let's start with **`za-idea`** to groom your idea into a concrete concept.

## 2. Platform Calibration

- **[SA]** I see you're using **{platform}**.
- **[SA]** Tech stack identified: **{stack}**.
- **[SA]** I've initialized the **Shared Board** and **Project Memory**.

## 3. Team Assembly

- **[SA]** I've assembled a specialized team for this project:
- **[PM]** John, **[BA]** Mary, **[ARCH]** Winston, **[DEV]** Amelia, **[DESIGN]** Sally, **[DB]** Atlas, **[QA]** Fiona, **[SEC]** Shield, **[OPS]** Deploy.

---

**[SA] Ready when you are!** What's the next move?
