## 🚀 The 7-Phase Dominance Pipeline

Aizen-Gate 2.0 enforces a rigorous SDLC to ensure zero-defect delivery:

1.  **`specify`** — Elite discovery interview via `[PM]` to create functional `spec.md`.
2.  **`research`** — Parallel technical research via 4 sub-agents into `research.md`.
3.  **`plan`** — Architectural debate between `[ARCH]` and `[DEV]` with XML decisions.
4.  **`tasks`** — Decomposition into Work Packages (WPs) with dependency mapping.
5.  **`auto`** — Autonomous wave execution. Spawns parallel worktrees per WP.
6.  **`implement`** — (Manual) Focused implementation of a specific WP.
7.  **`review`** — `[QA]` code review with Athena quality gates.
8.  **`merge`** — Topological merging of feature branches into main.
