# Aizen-Gate Workflows

## Overview

Workflows are reusable, multi-step sequences that choreograph several agents and skills.

## Core Workflows

### 1. The "OpenSpec" Loop

Go from a raw idea to a verified specification.

1. `sa-idea` (PM/BA)
2. `sa-discuss` (Architect/User)
3. `quality-gate` (QA/Architect)

### 2. The "Ralph" Implementation Loop

Autonomous continuous integration.

1. `sa-auto` (Developer/QA)
2. `sa-verify` (QA)
3. `sa-compound` (Scrum Master)

### 3. The "Mnemosyne" Mapping

For brownfield projects.

1. `sa-map` (Architect/Analyst)
2. `shared/project.md` creation
3. `sa-plan` (PM/Architect)
